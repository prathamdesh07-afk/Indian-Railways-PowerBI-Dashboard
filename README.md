# 🚆 Indian Railways Operations Dashboard (Power BI)

## 📊 Project Overview
This project presents an interactive Power BI dashboard analyzing Indian Railways operational data to generate meaningful insights.

## 🔹 Features
- Total trains analysis
- Average ticket fare insights
- Train distribution by type
- Interactive filters and visuals
- Clean dashboard UI design

## 🛠 Tools Used
- Power BI
- Excel
- Data Visualization
